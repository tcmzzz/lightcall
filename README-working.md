LightCall 旨在提供一个简单、高效且易于维护的外呼解决方案，适合小型团队或需要和内部系统集成的场景。

## 项目简介

LightCall 提供了一个完整的呼叫中心解决方案，包括：

- 外呼网关管理
- 电话号码管理
- 目标和任务管理
- 呼叫活动记录
- 与三方系统数据(目标/任务/活动)集成同步
- 云端服务集成
- 录音和通话记录管理

LightCall 是一个基于 Go 和 Vue3 的全栈项目，使用 PocketBase 作为后端框架, 集成了 FreeSWITCH 进行呼叫处理。
